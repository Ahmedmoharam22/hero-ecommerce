# 🛒 APEXSTORE - Modern E-Commerce Platform

APEXSTORE is a high-performance E-commerce frontend built with **Next.js 15**, **React 19**, and **TypeScript**. It features a seamless user experience with real-time state management and a polished UI.

## ✨ Features

- **Dynamic Product Routing**: SEO-friendly pages for every product.
- **Advanced State Management**: Powered by **Zustand** with persistent storage.
- **Cart System**: Real-time add/remove and quantity control.
- **Wishlist**: Save favorites with one click (Persistent).
- **Modern UI**: Built with **Shadcn UI**, **Tailwind CSS**, and **Framer Motion** for animations.
- **Data Fetching**: Optimized API calls using **React Query** (TanStack Query).
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: Framer Motion
- **State**: Zustand
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

📂 Project Structure
src/app: Application routes and pages.

src/features: Domain-driven components and logic (Products, Cart, Auth).

src/store: Global state management (Zustand).

src/components/ui: Shared UI components (Shadcn).

src/lib: Configuration for Axios and utilities.
